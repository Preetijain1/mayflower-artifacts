## Notes:
- `{{ class }}` is a local twig variable which is set in the atom twig template by appending `image.pull` to a CSS class name

### Variables
~~~
image {
    alt:
        type: string / required
    src:
        type: string (url) / required
    pull:
        type: string ("right" || "left")
}
~~~
